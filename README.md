# QazCorpBank — Фронтенд

## ⚡ Запуск

```bash
npm install
npm start
```
Откройте: **http://localhost:3001**

> ⚠️ Не открывайте index.html напрямую — авторизация не будет работать!

---

## 📱 SMS-верификация (DEV режим)

После ввода телефона в форме регистрации:
- Код появится во **всплывающем уведомлении** прямо на странице
- Также виден в **консоли сервера** (терминал где запущен npm start)
- Или введите универсальный код: **`000000`**

---

## 👥 Поток одобрения сотрудника

1. Сотрудник заполняет форму регистрации → подтверждает SMS → загружает документы → отправляет заявку
2. **Директор** входит в систему → раздел «Сотрудники» → видит карточки «На рассмотрении»
3. Нажимает «Просмотр и подтверждение» → выбирает роль → «Одобрить»
4. Сотрудник получает доступ — может войти по ИИН и паролю
